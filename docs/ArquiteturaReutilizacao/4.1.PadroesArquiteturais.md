# Documento de Arquitetura 
 
## 1. Introdução 
 
### 1.1 Objetivo 
 
Este documento tem como objetivo descrever os aspectos referentes à arquitetura do Projeto. Assim como referenciar e evidenciar as decisões arquiteturais que foram tomadas ao longo do desenvolvimento, assim como os padrões padrões que foram adotados. 
 
### 1.2 Escopo 
 
Este documento tem como finalidade apresentar a visão arquitetural do sistema desenolvido. Sendo assim descrever tecnicamente a solução adotada, tornando o priojeto manutenível e apresentando os padrões a serem utilizados ao longo do desenvolvimento. O documento será composto pela apresentação dos componentes de software, frameworks de desenvolvimento, padrões de software utilizados, casos de uso e persistência de dados.
